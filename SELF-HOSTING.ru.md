# Руководство по самостоятельному размещению

## Возможности

- **Куратированная библиотека промптов** — Доступ к 100+ высококачественным, проверенным сообществом промптам для ChatGPT, Claude, Gemini, Llama, Mistral и других AI-моделей
- **Обнаружение и просмотр** — Исследуйте промпты по категориям, тегам или через AI-семантический поиск
- **Создание и обмен промптами** — Отправляйте свои промпты с поддержкой текста, структурированных форматов (JSON/YAML) и медиа-расширенных форматов
- **Контроль версий** — Отслеживайте изменения промптов со встроенной системой версионирования и запросов на изменения (аналог PR)
- **Персонализированная лента** — Подписывайтесь на категории и получайте кураторскую ленту промптов, соответствующих вашим интересам
- **Приватные промпты** — Держите свои промпты приватными или делитесь ими с сообществом
- **Голосование и рейтинги** — Голосуйте за промпты и открывайте самые популярные через таблицу лидеров PromptMasters
- **Многоязычная поддержка** — Доступно на английском, испанском, японском, турецком, китайском и русском

## Преимущества

- **Раскройте потенциал AI:** Прекратите бороться с инженерией промптов — используйте проверенные в боях промпты от сообщества с 141k+ звезд на GitHub
- **Экономьте время:** Копируйте промпты одним кликом, настраивайте переменные в реальном времени и используйте их мгновенно в любом AI-чате
- **Качество от сообщества:** Каждый промпт курируется и улучшается сообществом через запросы на изменения и голосование
- **Самостоятельное размещение:** Разверните свою собственную white-label библиотеку промптов для команды или организации с настраиваемым брендингом, темами и аутентификацией
- **Лицензия CC0:** Все промпты находятся в общественном достоянии — используйте их свободно для любых целей, коммерческих или личных

## Начало работы

**Требования:**
- **План:** Бесплатный и с открытым исходным кодом (лицензия CC0)
- **Права пользователя:** Аккаунт не нужен для просмотра; войдите через GitHub/Google для создания и сохранения промптов
- **Доступность:** Общедоступен на [prompts.chat](https://prompts.chat)

---

Это руководство объясняет, как развернуть **prompts.chat** на вашем собственном приватном сервере для повышения конфиденциальности и кастомизации.

## Предварительные требования

- **Node.js** 18+ 
- База данных **PostgreSQL**
- **npm** или **yarn**

## Переменные окружения

Создайте файл `.env` на основе `.env.example`:

```bash
# База данных
DATABASE_URL="postgresql://user:password@localhost:5432/prompts"

# Аутентификация (выберите одного провайдера)
# GitHub OAuth
AUTH_GITHUB_ID="ваш-github-client-id"
AUTH_GITHUB_SECRET="ваш-github-client-secret"

# Или Google OAuth
AUTH_GOOGLE_ID="ваш-google-client-id"
AUTH_GOOGLE_SECRET="ваш-google-client-secret"

# NextAuth
AUTH_SECRET="сгенерируйте-случайный-секрет"

# Опционально: AI-семантический поиск
OPENAI_API_KEY="ваш-openai-api-key"
```

## Установка

### Быстрый старт (Рекомендуется)

Самый быстрый способ создать новый экземпляр prompts.chat:

```bash
npx prompts.chat new my-prompt-library
cd my-prompt-library
```

Это:
1. Клонирует чистую копию репозитория (без файлов разработки)
2. Установит зависимости
3. Запустит интерактивный мастер настройки

### Ручная установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/f/awesome-chatgpt-prompts.git
   cd awesome-chatgpt-prompts
   ```

2. **Установите зависимости**
   ```bash
   npm install
   ```

3. **Запустите интерактивный мастер настройки**
   ```bash
   npm run setup
   ```
   Это проведет вас через настройку:
   - **Брендинг** — Название приложения, логотип, описание
   - **Тема** — Основной цвет, радиус границ, вариант UI
   - **Аутентификация** — GitHub, Google, Apple, Azure AD или email/пароль
   - **Языки** — Выберите из 16 поддерживаемых локалей
   - **Функции** — Приватные промпты, категории, теги, комментарии, AI-поиск, AI-генерация, поддержка MCP

4. **Настройте переменные окружения**
   ```bash
   cp .env.example .env
   # Отредактируйте .env с вашими данными базы данных и аутентификации
   ```

5. **Запустите миграции базы данных**
   ```bash
   npm run db:migrate
   ```

6. **Заполните начальные данные** (опционально)
   ```bash
   npm run db:seed
   ```

7. **Запустите сервер разработки**
   ```bash
   npm run dev
   ```

8. **Сборка для продакшена**
   ```bash
   npm run build
   npm run start
   ```

## Конфигурация

Мастер настройки (`npm run setup`) автоматически генерирует `prompts.config.ts`. Вы также можете отредактировать его вручную:

```typescript
// Установите true для использования вашего брендинга вместо брендинга prompts.chat
const useCloneBranding = true;

export default defineConfig({
  // Брендинг
  branding: {
    name: "Ваша библиотека промптов",
    logo: "/ваш-логотип.svg",
    logoDark: "/ваш-логотип-темный.svg",
    description: "Ваше описание",
  },

  // Тема
  theme: {
    radius: "sm",        // "none" | "sm" | "md" | "lg"
    variant: "default",  // "flat" | "default" | "brutal"
    colors: {
      primary: "#6366f1",
    },
  },

  // Аутентификация
  auth: {
    provider: "github",  // "credentials" | "github" | "google" | "azure"
    allowRegistration: true,
  },

  // Функции
  features: {
    privatePrompts: true,
    changeRequests: true,
    categories: true,
    tags: true,
    aiSearch: false,  // Требует OPENAI_API_KEY
  },

  // Главная страница
  homepage: {
    useCloneBranding,  // Используйте свой брендинг на главной
    achievements: {
      enabled: !useCloneBranding,  // Скрыть достижения prompts.chat
    },
    sponsors: {
      enabled: !useCloneBranding,  // Скрыть спонсоров prompts.chat
    },
  },

  // Интернационализация
  i18n: {
    locales: ["ru", "en", "es", "ja", "tr", "zh"],
    defaultLocale: "ru",
  },
});
```

### Режим клонирования брендинга

Когда `useCloneBranding` установлен в `true`, главная страница будет:

- Отображать **название вашего брендинга** как заголовок hero
- Показывать **описание вашего брендинга** под заголовком
- Использовать ваш **логотип** как фоновый водяной знак вместо видео
- Скрывать кнопку "Развернуть ваш приватный сервер"
- Скрывать секцию достижений (Forbes, звезды GitHub и т.д.)
- Скрывать ссылки на спонсоров и призыв "Стать спонсором"

Это идеально подходит для организаций, которые хотят развернуть свою собственную white-label библиотеку промптов без брендинга prompts.chat.
